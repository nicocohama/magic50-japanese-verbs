# magic50-japanese-verbs
Magic 50 is a visual map designed to help learners and teachers understand Japanese verb forms as a system, not as isolated rules.  This repository contains the first public release of the Magic 50 map.

Version: 1.1 (First public release)  
Author: nicoco  
Year: 2025

This project is evolving.
